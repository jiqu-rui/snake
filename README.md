# 贪吃蛇 Snake Game

一个基于 HTML5 Canvas 的贪吃蛇网页游戏，支持多种主题风格和移动端适配。

## 功能特性

- **三种视觉主题**：经典、霓虹、极简风格一键切换
- **Canvas 渲染**：流畅的游戏画面和粒子特效
- **分数系统**：当前分数、最高分数实时展示（LocalStorage 持久化）
- **速度调节**：滑块实时调整游戏速度
- **游戏控制**：开始、暂停、重新开始，支持键盘和触屏操作
- **移动端适配**：虚拟方向键，支持手机和平板游玩
- **游戏统计**：蛇身长度、游戏时长、吃到的食物数量

## 操作方式

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → / W A S D | 控制蛇的移动方向 |
| 空格键 | 暂停 / 继续 |
| 触屏方向键 | 移动端方向控制 |

## 项目结构

```
snake/
├── snake.html        # 游戏主页面
├── snake-script.js   # 游戏逻辑（SnakeGame 类）
├── snake-style.css   # 样式与主题变量
└── .gitignore        # Git 忽略配置
```

## 快速开始

直接用浏览器打开 `snake.html` 即可开始游戏，无需构建工具或依赖安装。

## 技术栈

- HTML5 Canvas
- CSS3 自定义属性（主题切换）
- 原生 JavaScript（ES6 Class）
- Font Awesome 图标库（CDN）
